Editing JBang scripts and applications in an IDE

Since JBang 0.102 jbang edit [directory] will open the directory as if your IDE has a JBang plugin installed. This has the advantage that editing multiple files and especially related files 'just works'. You can still use jbang edit [filename] to have JBang generate a temporary project with symoblic linked files to edit in any IDE (but with some limits).

Below documents the known working IDE and their plugins.

Visual Studio Code mdi-microsoft-visual-studio-code IntelliJ IDEA Eclipse

JBang extension for Visual Studio Code

Available from Microsoft Marketplace and Open VSX Marketplace.

JBang edit command will offer to setup and install VSCodium (a free distribution of VSCode) if no other IDE found.

JBang Intellij Plugin

Available from JetBrains Marketplace.

Eclipse Eclipse Integration

Available from Eclipse Marketplace
